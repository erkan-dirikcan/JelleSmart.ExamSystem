Özellikler
Metronic 7 Tema Uzmanlığı - Sayfa yapıları, CSS/JS organizasyonu
ManageNavPages Entegrasyonu - Menü aktiflik durumları için hazır şablon
JavaScript Şablonu - KL prefix'li modüler yapı
Partial View Şablonları - Ana menü ve layout yapıları
SCSS/Color Palette - Tema özelleştirme rehberi
Authentication Sayfaları - Login/Register/Forgot Password şablonları
Kullanım
Bu skill, ASP.NET Core projelerinde Metronic 7 teması ile frontend geliştirme yaparken otomatik olarak devreye girecek. Şunları yapar:

Assets klasörünü wwwroot'a kopyalar
Logo ve favicon yerleştirir
ManageNavPages sınıfını günceller
JavaScript dosyalarını doğru yapıda oluşturur
Partial view'leri şablona göre hazırlar
AJAX/jQuery işlemlerini doğru şekilde uygular
Skill'i Claude Code'a yükleyerek kullanmaya başlayabilirsiniz.
